Dijkstra's algorithm is commonly used algorithm to determine the shortest path in a graph with non-negative weights with numerous applications in logistics or communications and social networks. 


```py
G : dict = {
    0: [(1, 10), (2, 10)],
    1: [(2, 10), (3, 20), (4, 20)],
    2: [(6, 40)],
    3: [(4, 20), (5, 15), (6, 40)],
    4: [],
    5: [(6, 10)],
    6: [(7, 10)],
    7: []
    }
```

The code below relies on the ```heapq``` module that is available in every Python distribution.

```py
def dijkstra(G : dict, s):
    D = {v: float('inf') for v in G.keys()}; D[s] = 0
    P = {v: None for v in G.keys()}
    q = [(D[s], s)]

    while q:
        _, v = heapq.heappop(q)
        for w, d in G[v]:
            if D[w] > D[v] + d:
                D[w] = D[v] + d
                P[w] = v
                heapq.heappush(q, (D[w], w))
    return D, P
```

An interesting application of Dijkstra's algorithm 


```py
import heapq

G : dict = {
    0: [(1, 10), (2, 10)],
    1: [(2, 10), (3, 20), (4, 20)],
    2: [(6, 40)],
    3: [(4, 20), (5, 15), (6, 40)],
    4: [],
    5: [(6, 10)],
    6: [(7, 10)],
    7: []
    }

"""
graph G:
    - keys: vertices
    - values: [(neighbor, distance)]
"""

def dijkstra(G : dict, s):

    D = {v: float('inf') for v in G.keys()}; D[s] = 0
    P = {v: None for v in G.keys()}
    q = [(D[s], s)]

    while q:
        _, v = heapq.heappop(q)

        for w, d in G[v]:
            if D[w] > D[v] + d:
                D[w] = D[v] + d
                P[w] = v
                heapq.heappush(q, (D[w], w))

    return D, P

print(dijkstra(G, 0))
```
